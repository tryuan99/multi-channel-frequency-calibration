# Multi-Channel Frequency Calibration

This repository contains various snapshots of the [openwsn-fw](https://github.com/tryuan99/openwsn-fw) repository.

The following branches contain different projects for SCuM and for the OpenMote related to frequency calibration.

### Bootstrap Calibration

- [`titan/channel_cal`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/channel_cal): Code for the OpenMote under [`projects/common/01bsp_channel_cal`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/channel_cal/projects/common/01bsp_channel_cal). The accompanying SCuM code can be found [here](https://github.com/tryuan99/scum-bootstrap-calibration/tree/main/scm_v3c/applications/channel_cal).

### Single-Channel Frequency Calibration

- [`titan/single_channel_cal_scum`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/single_channel_cal_scum): Code for SCuM can be found under [`projects/scum/03oos_openwsn`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/single_channel_cal_scum/projects/scum/03oos_openwsn)
- [`titan/single_channel_cal_openmote`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/single_channel_cal_openmote): Code for the OpenMote can be found under [`projects/common/03oos_openwsn`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/single_channel_cal_openmote/projects/common/03oos_openwsn)

### Multi-Channel Frequency Calibration

- [`titan/multi_channel_cal_scum`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/multi_channel_cal_scum): Code for SCuM can be found under [`projects/scum/03oos_openwsn`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/multi_channel_cal_scum/projects/scum/03oos_openwsn)
- [`titan/multi_channel_cal_openmote`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/multi_channel_cal_openmote): Code for the OpenMote can be found under [`projects/common/03oos_openwsn`](https://github.com/tryuan99/multi-frequency-calibration/tree/titan/multi_channel_cal_openmote/projects/common/03oos_openwsn)
